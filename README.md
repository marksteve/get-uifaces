Script to download [User Inter Faces](http://uifaces.com) urls

## Install

```
git clone https://github.com/marksteve/get-uifaces.git
cd get-uifaces
virtualenv venv
venv/bin/pip install -r requirements.txt
venv/bin/python main.py <count>
```
